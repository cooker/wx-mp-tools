微信公众号管理 Chrome 插件 - 隐私政策

生效日期：2025 年 2 月  |  适用产品：微信公众号管理 Chrome 插件（WeChat Public Account Manager）

一、概述

本插件（以下简称「本产品」）尊重并保护用户隐私。本隐私政策说明我们如何对待您的数据，请在使用前阅读。

核心承诺：本产品不向开发者或任何第三方服务器上传、收集或发送您的个人身份信息或公众号内容数据。

二、我们收集的数据

2.1 我们不收集的数据

  • 本产品不收集您的姓名、邮箱、手机号、微信账号等个人身份信息。
  • 本产品不收集您公众号内的文章内容、标题、图片等业务数据用于任何外部服务器。
  • 本产品不将您的数据用于分析、广告或向第三方出售、共享。

2.2 仅在您设备本地保存的数据

以下数据仅保存在您的浏览器本地（Chrome 扩展本地存储或本地存储），不会上传到我们的服务器：

数据类型用途说明存储位置
图文模板您创建的公众号名称、标题、正文模板浏览器本地存储 (localStorage)
插件配置悬浮框位置、通知开关、主题、API 等设置Chrome 扩展本地存储 (chrome.storage.local)
历史插入记录用于「往期推荐」等功能的本地记录浏览器本地存储 (localStorage)

上述数据仅用于在本插件内提供您所选择的功能（如模板填充、批量打开、往期推荐等）。

三、数据的使用方式

3.1 本地功能

  • 模板与自动填充：您保存的模板仅在您访问微信公众平台（mp.weixin.qq.com）时，于本插件内部用于自动填充标题和内容，数据不离开您的设备。
  • 批量操作与文章列表：本插件会从当前页面或 Cookie 中读取 token、timestamp、lang 等参数，仅用于在您已登录的会话下打开微信公众平台页面或调用其接口，以完成批量打开、文章列表等功能。这些请求直接发往微信公众平台，不经过我们的服务器。

3.2 可选 API 配置

在「选项」页面中，您可自愿填写「API 端点」和「API 密钥」。若您填写并保存，这些信息仅保存在您的设备上；若本产品在功能上调用您配置的端点，则请求将发往您指定的地址，我们不会接收或存储您通过该 API 传输的任何数据。

3.3 网络访问范围

本插件仅请求访问 https://mp.weixin.qq.com/ 及其子路径，用于在微信公众平台页面内提供功能。我们不会代表您向其他域名发送您的个人或公众号数据。

四、数据的存储与保留

  • 所有上述数据均存储在您使用本插件的设备（及浏览器配置文件）上。
  • 数据保留时间与您的使用周期一致：卸载本插件或清除扩展数据后，Chrome 会清除扩展的本地存储；清除网站数据时，可能影响保存在该域名下的本地存储数据。

五、数据安全

  • 我们无法访问您设备上的本地数据，因此不会对您本地存储的数据进行主动「安全处理」或备份。
  • 建议您仅在可信设备上使用本插件，并妥善保管您的微信公众平台账号与密码；本产品不存储您的登录凭据。

六、儿童与敏感信息

  • 本产品不面向儿童主动收集个人数据。
  • 请您勿在模板或本插件中保存敏感个人信息（如密码、身份证号、银行卡号等）；若您选择保存，相关风险由您自行承担。

七、隐私政策的变更

  • 我们可能会适时更新本隐私政策，更新后的版本会在本页面（或您获取本政策的同一位置）发布，并注明生效日期。
  • 若变更涉及对您已存储数据的用途或共享方式有重大影响，我们会在产品内或发布说明中尽量提示。

八、联系与权利

  • 您的权利：您可随时卸载本插件以停止使用，并可通过浏览器清除扩展数据或网站数据来删除上述本地数据。
  • 联系我们：若您对本隐私政策或数据处理方式有疑问，请通过本插件的发布页面（如 Chrome 网上应用店）中的开发者联系方式与我们联系。

感谢您使用微信公众号管理 Chrome 插件。
