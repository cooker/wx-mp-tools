微信公众号管理 Chrome 插件 - 隐私政策

生效日期：2025 年 2 月  |  适用产品：微信公众号管理 Chrome 插件（WeChat Public Account Manager）

一、概述

本隐私政策用于说明：当您使用本插件时，我们如何处理与您相关的本地数据与网络请求。 在您开始使用前，请您仔细阅读本政策。

核心承诺：本插件不向我们的服务器上传您的个人身份信息或公众号内容数据。

二、数据收集与存储

2.1 我们不收集的数据

  • 不收集您的姓名、邮箱、手机号、微信账号等个人身份信息。
  • 不将您公众号的文章内容、标题、图片等业务数据上传至外部服务器。
  • 不使用您的数据进行分析、广告投放或向第三方出售/共享。

2.2 仅保存在您设备本地的数据

以下数据仅保存在您的浏览器本地存储（例如 localStorage 或 Chrome 扩展本地存储）， 不会上传到我们的服务器：

数据类型 用途说明 存储位置
图文模板 您创建的公众号名称、标题、正文模板 localStorage
插件配置 悬浮框位置、通知开关、主题、API 等设置 chrome.storage.local
历史插入记录 用于「往期推荐」等功能的本地记录 localStorage

三、数据的使用方式

  • 模板与自动填充：您保存的模板仅在您访问微信公众平台（mp.weixin.qq.com）时， 于本地用于自动填充标题和内容，不离开您的设备。
  • 批量操作与文章列表：插件会使用您已登录会话相关信息（例如页面中读取到的参数）， 直接与微信公众平台交互以完成页面跳转或功能调用；请求不会经过我们的服务器。
  • 可选 API 配置：若您在插件的「选项」中填写 API 端点与密钥， 这些信息仅保存在您的设备上；当插件调用您配置的端点时，请求将发送到您指定的地址。

四、网络请求与第三方网站

本插件主要访问 https://mp.weixin.qq.com/ 及其子路径以实现相关功能。 插件可能包含指向第三方网站的链接；第三方网站的内容与隐私政策由其自身负责。

五、数据保留与删除

  • 本插件相关数据保存在您的设备上，保留时间与您的使用周期一致。
  • 卸载插件或清理扩展数据后，Chrome 可能会清除插件本地存储。
  • 您也可以通过浏览器清除网站数据来删除本地存储内容。

六、数据安全

  • 我们无法访问您的本地存储内容，因此不会对本地数据进行主动备份或迁移。
  • 建议您在可信设备上使用本插件，并妥善保管微信公众平台账号与密码。

七、儿童与敏感信息

  • 本插件不面向儿童主动收集个人数据。
  • 请勿在模板或插件中保存敏感个人信息（如密码、身份证号、银行卡号等）。

八、隐私政策的变更

  • 我们可能会更新本隐私政策。更新后的版本会发布在本页面，并注明生效日期。
  • 若变更对您已存储数据的用途产生重大影响，我们将尽量在产品内或发布说明中提示。

九、联系与权利

  • 您的权利：您可随时卸载本插件，并通过浏览器清除扩展数据或网站数据删除本地内容。
  • 联系我们：如对本隐私政策有疑问，请通过本插件在 Chrome 网上应用店的发布页面提供的开发者联系方式联系。
