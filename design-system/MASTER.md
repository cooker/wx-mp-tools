# 工具导航 · 设计系统 v2

基于 UI UX Pro Max 匹配结果，结合 **Neumorphism（新拟态）+ 搜索 / 筛选 / 收藏 + 移动端适配**。

---

## 1. 设计方向

| 维度 | 选择 | 说明 |
|------|------|------|
| 主风格 | Neumorphism | 软 UI、浮雕/凹陷、凸面/凹面、柔和阴影、圆角 |
| 新增功能 | 搜索、分类筛选、收藏 | 见第 3 节 |
| 适配 | 移动端优先 | 375px、768px、1024px 断点 |

---

## 2. 色彩系统（Neumorphism）

**来源：** Neumorphism 风格 + 提升对比度（兼顾无障碍）

| 角色 | Hex | 用途 |
|------|-----|------|
| 背景 | `#E8E8E8` | 主背景（浅灰，光线假设来自左上） |
| 凸面/凸起 | `#F5F5F5` | 凸起元素（亮边在左上） |
| 凹面/按下 | `#D8D8D8` | 凹陷、输入框、按下态 |
| 正文 | `#2D2D2D` | 主文字，保证 4.5:1 对比 |
| 次要文字 | `#5A5A5A` | 描述、辅助 |
| 强调色 | `#6B8DD6` | 链接、选中、焦点（Soft Blue） |

**阴影约定（光线左上 45°）：**

- **凸起：** `box-shadow: -5px -5px 15px #fff, 5px 5px 15px rgba(0,0,0,0.1)`
- **凹陷：** `box-shadow: inset 3px 3px 6px rgba(0,0,0,0.1), inset -3px -3px 6px #fff`
- **按下：** 阴影减弱，类似轻微凹陷

**注意：** Neumorphism 本身对比度偏低，需确保文字与背景至少 4.5:1。

## 3. 功能模块

### 3.1 搜索

| 项目 | 说明 |
|------|------|
| 位置 | Header 下方，主内容上方，全宽或与筛选栏并排 |
| 形态 | 圆角输入框，Neumorphism 凹陷样式 |
| 交互 | 输入时 debounce 后实时过滤（约 300ms） |
| 空态 | 显示「无结果」并给出建议文案（如「试试其他关键词」） |
| 移动端 | 可折叠为图标，点击展开输入框 |

### 3.2 分类筛选

| 项目 | 说明 |
|------|------|
| 位置 | 搜索框下方或同排右侧 |
| 形态 | 胶囊 / 标签按钮，凸起样式，选中态为凹陷 |
| 逻辑 | 全部 | 开发工具 | 公众号 | 实用工具 …（与 nav.config 分类对应） |
| 交互 | 点击切换，配合搜索共同过滤 |
| 移动端 | 横向滚动或折叠为下拉 |

### 3.3 收藏

| 项目 | 说明 |
|------|------|
| 存储 | `localStorage`，key 如 `tool-nav-favorites` |
| 形态 | 工具卡片右上角星形图标，未收藏为空心，已收藏为实心 |
| 入口 | 可选：Header 旁「我的收藏」入口，仅展示已收藏项 |
| 交互 | 点击切换收藏态，过渡 150ms |
| 移动端 | 星形尺寸 ≥ 44×44px 触摸区 |

## 4. 布局结构

```
┌─────────────────────────────────────────────────────────┐
│  Header（标题 + 描述）                                    │
├─────────────────────────────────────────────────────────┤
│  [ 搜索框 ]                    [ 我的收藏 ]  （可选）      │
├─────────────────────────────────────────────────────────┤
│  [ 全部 ] [ 开发工具 ] [ 公众号 ] [ 实用工具 ]  （筛选）   │
├───────────────────────────────┬─────────────────────────┤
│  工具卡片网格                   │  右侧公告（桌面端）       │
│  ┌─────┬─────┬─────┐          │  ┌─────────────────────┐ │
│  │ ⭐  │ ⭐  │ ⭐  │          │  │ 公告列表             │ │
│  ├─────┼─────┼─────┤          │  └─────────────────────┘ │
│  │ ⭐  │ ⭐  │ ⭐  │          │                         │
│  └─────┴─────┴─────┘          │                         │
├───────────────────────────────┴─────────────────────────┤
│  底部广告位                                              │
└─────────────────────────────────────────────────────────┘
```

**移动端（<768px）：**

- 搜索 + 筛选在上方，可纵向堆叠
- 公告区移至主内容上方或可折叠
- 工具卡片单列或 2 列，触摸目标 ≥ 44×44px

## 5. 排版

| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| 站点标题 | Nunito Sans / Varela Round | 1.5rem–2rem | 700 |
| 分类标题 | 同正文 | 0.95rem | 600 |
| 工具名称 | Nunito Sans | 0.95rem | 500 |
| 描述 | Nunito Sans | 0.85rem | 400 |
| 正文 | Nunito Sans | 1rem | 400 |

**原则：** 圆角字族配合 Neumorphism，保持柔和一致。

## 6. 组件规范（Neumorphism）

### 6.1 搜索框

- 背景：与主背景同色或略深，凹陷阴影
- 圆角：`12px` 或 `16px`
- 内边距：`0.75rem 1rem`
- 占位符：`搜索工具名称或描述…`
- Focus：凸起或边框高亮（强调色）

### 6.2 筛选标签

- 未选中：凸起阴影
- 选中：凹陷阴影或浅色填充
- 圆角：`20px`（胶囊形）
- Hover：阴影略减弱，过渡 150ms

### 6.3 工具卡片

- 凸起样式，圆角 `12px` 或 `16px`
- 收藏图标：右上角，SVG，可点击区域 ≥ 44×44px
- 按下态：可选用凹陷或阴影减弱
- Hover：阴影略微加强

### 6.4 公告卡片

- 与工具卡片一致的凸起样式
- 圆角、阴影风格统一

### 6.5 广告位

- 与整体 Neumorphism 风格统一，避免强对比或花哨

## 7. 动效与交互

| 类型 | 时长 | 说明 |
|------|------|------|
| 按下 | 150ms | Neumorphism 软按反馈 |
| Hover | 200ms | 阴影、透明度微调 |
| 筛选切换 | 150ms | 标签状态变化 |
| 收藏切换 | 150ms | 星形填充动画 |

**原则：** 支持 `prefers-reduced-motion: reduce`，必要时禁用动效。

---

## 8. 移动端规范

| 项目 | 要求 |
|------|------|
| 视口 | `width=device-width, initial-scale=1` |
| 触摸目标 | ≥ 44×44px |
| 搜索 | 可收缩为图标，点击展开 |
| 筛选 | 横向滚动或折叠下拉 |
| 输入法 | 搜索框使用 `inputmode="search"` |
| 断点 | 375px、768px、1024px |

---

## 9. 避免事项（Anti-patterns）

- ❌ 使用 emoji 作图标，改用 SVG
- ❌ 文字与背景对比度不足（至少 4.5:1）
- ❌ 移动端点击区域过小
- ❌ 搜索无结果时空白或仅显示「0 条」
- ❌ 收藏无本地持久化

---

## 10. 实施优先级

1. **P0：** 色彩与 Neumorphism 阴影变量、工具卡片新拟态样式

3. **P2：** 收藏（localStorage、星形交互）
4. **P3：** 公告、广告位与移动端布局统一
